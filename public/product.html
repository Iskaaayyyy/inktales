InkTales Mobile App

Read. Write. Convert. Earn.

InkTales is the product platform of InkVerse. It uses a neon comic-inspired interface for readers and writers who want to publish stories, top up coins, apply for monetization, and convert published stories into text-accurate manga panels, comic pages, or animated video previews.

Product features.

InkTales has one login for all users. A user can be both reader and writer.

📝

Unified Writing and Reading

One user account can write stories, read stories, publish works, unlock paid stories, like, tip, and report content.

🎞

Story Conversion

Published stories can be converted into text-accurate manga panels, comic pages, and animated video previews inside the installable mobile app.

🪙

Coins and Monetization

Users can top up coins, unlock paid stories, tip writers, apply for monetization, and request earnings withdrawals.

💎

Monetization Application

Writers can apply after meeting platform requirements for published stories, views, and likes.

🛡

Admin Operations

Admins manage users, top ups, withdrawals, reports, story library, monetization requests, and transaction records.

📱

Installable Mobile App

InkTales runs as a mobile-ready PWA and includes an Android APK build folder so the deployed system can offer a downloadable APK file.

🚩

Reports and Moderation

Readers can report stories while admins can review, resolve, feature, unpublish, or delete content.

App download.

Build the Android APK from the included Android project and place it in public/downloads/InkTales.apk so this system can serve it after deployment.
